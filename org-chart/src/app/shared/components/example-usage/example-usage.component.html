Reusable Table Component Demo

This demonstrates the capabilities of the reusable table component with various features.

Features Demonstrated:

  • Column Management: Toggle columns, sort by columns, filter by columns
  • Column Resizing: Drag the edge of column headers to resize columns
  • Horizontal Scrolling: Scroll horizontally when columns exceed container width
  • Global Search: Search across all visible columns
  • Row Selection: Select individual rows or all rows
  • Bulk Actions: Perform actions on multiple selected rows
  • Row Actions: Context-specific actions for each row
  • Pagination: Navigate through large datasets
  • Export: Export data to CSV format
  • Responsive Design: Adapts to different screen sizes
  • Custom Formatting: Format data like dates, currency, and status indicators
  • Row Classes: Apply CSS classes based on row data
  • Empty State: Handle cases when no data is available
